# HTML 단독 배포 테스트

이 브랜치는 `dashboard.html` 하나를 브라우저에서 직접 열어 사용하는 방식입니다.

## 실행

```text
dashboard.html
```

파일을 더블클릭해서 최신 Chrome 또는 Edge로 엽니다.

## 사용 방법

1. `Open` 버튼을 누릅니다.
2. `2605` 같은 `YYMM` 형식의 월별 데이터 폴더를 선택합니다.
3. 비교가 필요하면 `비교월 폴더 추가`로 다른 월 폴더를 추가합니다.

## 특징

- Python 서버를 실행하지 않습니다.
- 외부 CDN이나 인터넷 통신을 사용하지 않습니다.
- 선택한 로컬 `.xlsx` 파일을 브라우저 안에서 직접 읽습니다.

## 제한

- 최신 Chrome 또는 Edge의 `DecompressionStream` 지원이 필요합니다.
- 서버 버전의 디스크 캐시는 사용하지 않으므로, 같은 폴더를 다시 열 때도 다시 분석합니다.
- 매우 큰 데이터셋은 EXE 방식보다 느릴 수 있습니다.
