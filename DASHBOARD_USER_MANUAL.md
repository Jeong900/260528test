# DS 정기기부 대시보드 사용설명서

## 1. 대시보드 소개

DS 정기기부 대시보드는 월별 Excel 데이터를 읽어 정기기부 현황을 확인하는 도구입니다.

확인할 수 있는 화면은 다음과 같습니다.

- 달력 현황
- 사업부별 비교
- 기간별 비교
- Excel/CSV 다운로드
- 그래프 이미지 저장

대시보드는 외부 서버에 데이터를 보내지 않습니다. 사용자의 PC에 있는 Excel 파일을 로컬에서 읽어 분석합니다.

## 2. 실행 전 준비

EXE 파일과 월별 데이터 폴더를 같은 위치에 둡니다.

예시:

```text
DS_Dashboard.exe
2605\
2604\
2603\
```

월별 데이터 폴더 이름은 반드시 `YYMM` 형식이어야 합니다.

예시:

```text
2605 = 2026년 5월
2604 = 2026년 4월
2603 = 2026년 3월
```

## 3. Excel 파일명 규칙

각 월별 폴더 안의 Excel 파일명은 아래 형식을 사용합니다.

```text
YYMMDD_데이터(전체).xlsx
YYMMDD_데이터(전체)_dummy.xlsx
```

예시:

```text
260501_데이터(전체).xlsx
260502_데이터(전체).xlsx
260528_데이터(전체).xlsx
```

파일명이나 폴더명이 규칙과 다르면 대시보드에서 해당 파일을 읽지 못할 수 있습니다.

## 4. 실행 방법

1. `DS_Dashboard.exe`를 더블클릭합니다.
2. 브라우저가 자동으로 열립니다.
3. 첫 화면에서 `Open` 버튼을 누릅니다.
4. 분석할 월별 폴더를 선택합니다. 예: `2605`
5. 로딩이 끝나면 대시보드 화면이 표시됩니다.

브라우저 주소창에 아래와 같은 주소가 보일 수 있습니다.

```text
http://127.0.0.1:8765
```

이 주소는 외부 인터넷 주소가 아니라, 실행 중인 PC 내부 주소입니다.

## 5. 화면별 사용법

### 달력 현황

선택한 월의 일자별 데이터를 달력 형태로 보여줍니다.

상단에서 아래 항목을 바꿀 수 있습니다.

- 사업부
- 표시 지표

표시 지표:

- 참여인원
- 기부금액
- 평균 기부액

색이 진할수록 해당 지표 값이 상대적으로 큰 날짜입니다.

`Excel 다운로드` 버튼을 누르면 현재 달력 기준 데이터를 Excel 파일로 저장할 수 있습니다.

### 사업부별 비교

사업부별 추이를 그래프로 비교하는 화면입니다.

사용 방법:

1. 왼쪽 메뉴에서 `사업부별 비교`를 선택합니다.
2. 시작일과 종료일을 선택합니다.
3. 비교할 사업부를 체크합니다.
4. 그래프와 표를 확인합니다.

사용 가능한 버튼:

- `이미지 저장`: 그래프를 PNG 이미지로 저장
- `Excel 다운로드`: 사업부별 비교 데이터를 저장

### 기간별 비교

여러 월의 같은 기간을 비교하는 화면입니다.

사용 방법:

1. 먼저 기준 월 폴더를 `Open`으로 불러옵니다.
2. 왼쪽 메뉴에서 `기간별 비교`를 선택합니다.
3. `비교월 폴더 추가` 버튼을 누릅니다.
4. 비교할 다른 월 폴더를 선택합니다. 예: `2604`, `2603`
5. 시작일과 종료일을 선택합니다.
6. 보기 형식을 선택합니다.

보기 형식:

- 선그래프 형식
- 달력 형식

`Excel 다운로드` 버튼을 누르면 현재 선택한 기간 비교 표를 저장합니다.

## 6. 다운로드 파일

대시보드에서 다운로드하는 파일은 브라우저의 기본 다운로드 폴더에 저장됩니다.

파일명에는 보통 아래 정보가 포함됩니다.

- 기준 월
- 화면 이름
- 사업부
- 지표

예시:

```text
2605_달력현황_DS부문 전체_참여인원.xls
2605_사업부별비교.csv
2605_기간비교표_DS부문 전체_기부금액.xls
```

## 7. 자동 종료 안내

대시보드는 보안과 PC 자원 정리를 위해 자동 종료 기능이 있습니다.

아래 활동이 15분 동안 없으면 대시보드 프로세스가 자동 종료됩니다.

- 클릭
- 키보드 입력
- 마우스 이동
- 스크롤
- 화면 조작

자동 종료되면 캐시 폴더도 함께 삭제됩니다.

다시 사용하려면 `DS_Dashboard.exe`를 다시 실행하면 됩니다.

## 8. 자주 발생하는 문제

### 브라우저가 열리지 않는 경우

`DS_Dashboard.exe`를 다시 실행해 봅니다.

그래도 열리지 않으면 브라우저 주소창에 아래 주소를 직접 입력합니다.

```text
http://127.0.0.1:8765
```

### 폴더를 선택했는데 데이터가 나오지 않는 경우

아래를 확인합니다.

- 폴더명이 `2605`처럼 `YYMM` 형식인지
- Excel 파일명이 `260501_데이터(전체).xlsx` 형식인지
- Excel 파일이 월별 폴더 바로 아래에 있는지
- 파일이 열려 있거나 손상되어 있지 않은지

### 비교월을 추가했는데 표시되지 않는 경우

비교월 폴더명과 파일명이 규칙에 맞는지 확인합니다.

예:

```text
2604\
260401_데이터(전체).xlsx
260402_데이터(전체).xlsx
```

### 대시보드가 갑자기 꺼진 경우

15분 동안 사용하지 않아 자동 종료되었을 수 있습니다.

`DS_Dashboard.exe`를 다시 실행하면 됩니다.

### 같은 월을 다시 열었는데 처음보다 느린 경우

자동 종료 시 캐시가 삭제될 수 있습니다. 이 경우 다시 분석하므로 시간이 걸릴 수 있습니다.

## 9. 배포할 때 전달할 것

동료에게 전달할 때는 아래처럼 묶어서 전달합니다.

```text
DS_Dashboard.exe
2605\
2604\
2603\
```

실제 업무 데이터가 포함되어 있다면 회사 보안 정책에 맞게 전달합니다.

## 10. 보안 안내

- 대시보드는 선택한 로컬 Excel 파일을 PC 안에서 분석합니다.
- 데이터를 외부 서버로 업로드하지 않습니다.
- `127.0.0.1` 주소는 사용자의 PC 내부 주소입니다.
- 실제 데이터 폴더와 Excel 파일은 GitHub에 올리지 않습니다.
