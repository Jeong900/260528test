# EXE 배포 테스트

이 브랜치는 로컬 PC 안에서만 동작하는 EXE 배포 방식입니다.

## 빌드

```powershell
.\build_exe.ps1
```

빌드 결과:

```text
dist\DS_Dashboard.exe
```

## 동료에게 전달할 파일 구조

```text
DS_Dashboard.exe
2605\
2604\
2603\
```

EXE를 실행하면 `http://127.0.0.1:8765` 로컬 대시보드가 자동으로 열립니다. 이 주소는 외부 서버가 아니라 실행한 PC 자기 자신입니다.

콘솔창은 표시되지 않습니다.

사용자 입력, 마우스 이동, 스크롤, 클릭 등 활동이 15분 동안 없으면 EXE 프로세스가 자동 종료되고 `.dashboard_cache` 폴더가 삭제됩니다.

## 데이터 규칙

- 폴더명: `YYMM` 형식, 예: `2605`
- 파일명: `YYMMDD_데이터(전체).xlsx` 또는 `YYMMDD_데이터(전체)_dummy.xlsx`
- 데이터 폴더는 EXE와 같은 위치에 둡니다.
